Bisa Ceritakan Tentang Profile Diri Anda?


Rumus 1;

"Ceritakan Tentang Pendidikan"
Ceritakan nama + pendidikan terakhir + kegiatan + aktif disekolah/kampus + pengalaman yang dimiliki saat sekolah / kuliah


Nama saya ..., saya lulusan dari Universitas ..., fakultas ..., program study ...,

Selama kuliah, saya aktif berorganisasi dan aktif dalam mengikuti kegiatan kampus

Saat bergabung dalam organisasi, saya sering menjadi bendahara/sekertaris kegiatan untuk mengelola keuangan, mencatatat data & memasukkan data absensi, membuat laporan kegiatan, persentasi yang menggunakan Microsoft word, excel, powerpoint, dan project library. 



Rumus 2
"Ceritakan Tentang Pengalaman"
Ceritakan nama + pengalaman kerja di perusahaan sebelumnya + sampaikan jobsdesk secara detail + ceriatakan keahlian / kemampuan yang kamu kuasai


Nama saya ..., saya pernah bekerja di perusahaan ..., posisi sebagai ...,

jobdesk saya saat bekerj di perusahan ..., saya fokus membuat iklan promosi digital, seperti di facebook, instagra, youtube, dan terakhir belajar mengenai marketing TikTok

selama menjadi desain grafis, saya sering menggunakan software adobe photoshop, dan adobe ilustrator untuk menunjang performa design yang saya buat.


Rumus 3
"Ceritakan Kemampuan / Keahlian"
ceritakan nama + kegiatan yang saat ini masih dijalani + kemampuan / keahlian yang saat ini kamu miliki


Nama saya ..., kegiatan aktif saya saat ini sering menjadi MC / Moderator dikegiatan seminar, webinar, dan MC rapat diberbagai acara perusahaan

Saya memiliki kemampuan berkomunikasi yang baik dan lancar, serta menguasai dalam menggunakan komputer, saya juga terbiasa membuat laporan menggunakan Microsoft word, dan membuat laporan keuangan menggunakan Microsoft exel
